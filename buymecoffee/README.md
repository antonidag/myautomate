# buymecoffee

## Overview
Automatically logs in to buymecoffee.com and creates a new post
## Stack
- Node JS
- Shell JS
- Puppeteer JS

## Getting Started
### Prerequisites
- Git
- Node
- npm

Set up pipeline to trigger and run on http request.

## Installation

Set up .env 
```
BUY_ME_COFFEE_PASSWORD=
BUY_ME_COFFEE_USERNAME=
ENVIRONMENT= [DEV | PROD]
```
Install node packages:
```
npm install
```

Run locally:
```
npm start
```
